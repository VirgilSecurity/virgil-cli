NAME

container-confirm -- confirm password token and re-encrypt Private Key data with the new password.

SYNOPSIS

virgil container-confirm [−−confirmation-token arg]

DESCRIPTION

Confirm password token and re-encrypt Private Key data with the new password.

OPTIONS

−t arg, −−confirmation-token arg Confirmation token received on email box.
