NAME

container-reset -- reset container password.

SYNOPSIS

virgil container-reset [−−user-id arg] [−−container-pwd arg]

DESCRIPTION

Reset container password.
Note: A user can reset their Private Key object password if the
Container Type

equals ’easy’. If the Container Type equals ’normal’, the Private Key object will be stored in its original form.

OPTIONS

−u arg −−user-id arg User's identifer.

Format: [email|phone|domain]:<value>
where:

•

if email, then − user's email;

•

if phone, then − user's phone;

•

if domain, then − user's domain.

−c arg, −−container-pwd arg

Container password.
