NAME

public-key-update −− update user’s public key on the Virgil Public Keys service.

SYNOPSIS

virgil public-key-update [−−in file] [−−public-key-id arg] [−−private-key file] [−−private-pwd arg] [−−new-private-key file] [−−new-private-pwd arg]

DESCRIPTION

Update user’s public key on the Virgil Public Keys service.

OPTIONS

−i file, −−in file New Public key. If omitted stdin is used.
−e arg, −−public-key-id arg

Recipient's Virgil Public Key.

Format:[public-id|email|phone|domain]:<value>
where:

•

if public-id, then − user's public-key-id;

•

if email, then − user's email;

•

if phone, then − user's phone;

•

if domain, then − user's domain.

−k file, −−private-key file

Sender's old private key.

−p arg, −−private-pwd arg

Sender's old private key password.

−−new-private-key file

Sender's new private key.

−−new-private-pwd arg

Sender's new private key password.
