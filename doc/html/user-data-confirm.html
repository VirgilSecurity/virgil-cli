NAME

user-data-confirm -- send confirmation code to the Virgil Public Keys service. Confirmation code is sent after user’s public key registration.

SYNOPSIS

virgil user-data-confirm [−−user-data-id arg] [−−confirmation-code arg]

DESCRIPTION

Send confirmation code to the Virgil Public Keys service. Confirmation code is sent after user’s public key registration.

OPTIONS

−i arg −−user-data-id arg User's identifer.
−c arg, −−confirmation-code arg

Confirmation code.
