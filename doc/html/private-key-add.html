NAME

private-key-add −− load an existing Private Key into the Private Keys Service and associate it with the existing Container object. To register, you must provide a public key/s which will be associated with it.

SYNOPSIS

virgil private-key-add [−−in file] [−−private-pwd arg]] [−−public-key-id arg]

DESCRIPTION

Load an existing Private Key into the Private Keys Service and associate it with the existing Container object.

OPTIONS

−i file, −−in file Sender’s private key. If omitted stdin is used.
−p arg, −−private-pwd arg

Sender's private key password.

−e arg, −−public-key-id arg

Sender's Virgil Public Key.

Format:[public-id|email|phone|domain]:<value>
where:

•

if public-id, then − user's public-key-id;

•

if email, then − user's email;

•

if phone, then − user's phone;

•

if domain, then − user's domain.
